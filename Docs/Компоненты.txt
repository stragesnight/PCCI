Комплектующие ПК : 

Скорость работы компьютера зависит от всех его комплектующих

Материнская плата (МП) - печатная плата, предназначенная для подключения основных комплектующих компьютера, обеспечивает их работу и взаимодействие между собой. 
Характеристика : форм-фактор ; сокет ; чипсет. Обязательна

Процессор - микросхема и мозг компьютера, отвечает за выполнение всех операций, чем лучше процессор, тем быстрее выполняются процессы на компьютере, устанавливается на специальное место на МП. 
Характеристика : Intel или AMD; рабочая частота ; кол-во ядер; охлаждение. Обязательный с встроенным графическим ядром
Охлаждение процессора обязательно

Оперативная память - предназначена для временного хранения данных, данные с неё при выключении компьютера полностью стираются. 
Характеристика : тип ; потребление ; объем/планку ; частота. Обязательная

Жесткий диск - предназначен для длительного хранения данных. 
Характеристика : объем ; форм-фактор ; тип ; скорость вращения диска (обороты в минуту) ; объем буфера. Обязательный

Блок питания - обеспечивает все комплектующие необходимым количеством электроэнергии. 
Характеристика : тип ; мощность ; КПД (коэффициент полезного действия) ; разъемы. Обязательный

Корпус - позволяет аккуратно организовывать и расставлять все комплектующие, закручивая,прикручивая и подключая их для работы, в корпусе проще поддерживать необходимую температуру, 
он защищает комплектующие от повреждений. Характеристика : форм-фактор, организация охлаждения. Желательно

Видеокарта (или графическая плата) - предназначена для вывода графических данных на экран монитора и просчета сложных 3-D объектов, устанавливается в специальный разъем на МП PCI. 
Характеристика : Nvidia или AMD ; графический процессор ; видеоконтроллер ; память ; система охлаждения ; производительность. Необязательная

Звуковая карта - выводит звук на колонки, обычно встроена в материнскую плату, можно купить другую звуковую плату, если встроенная не устраивает. Необязательная

Сетевая карта - необходима для подключения компьютера в локальную сеть или в интернет. Характеристика : скорость принятия интернета (МБ/с, Мб/с и т.д.). Необязательна
