# PCCI - PC Component Info

Это репозиторий группового проекта команды №3 из ПВ-011.
В нём размещён исходный код проекта, а также его документация.

## О программе

Программа будет представлять собой интерактивный набор информации о
комплектующих персонального компьютера, а также об отдельных моделях
для каждой категории.
С её помощью пользователь сможет ознакомиться со строением компьютера,
характеристиками, особенностями и фактами о каждой его составляющей.
